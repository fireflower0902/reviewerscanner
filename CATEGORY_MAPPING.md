# 카테고리 매핑 규정 (Category Mapping Rules)

본 문서는 리뷰어스캐너 시스템에서 각 체험단 플랫폼(강남맛집, 리뷰플레이스, 리뷰노트)의 원천 카테고리 데이터를 내부 표준 14종 카테고리로 변환하는 규칙을 설명합니다.

## 표준 14종 카테고리
서비스 프론트엔드 및 데이터베이스에서 공통으로 사용되는 카테고리 목록입니다.
> **음식점, 뷰티, 여행, 숙박, 문화, 배달, 운동, 레저, 생활, 포장, 반려동물, 식품, 유아동, 기타**

---

## 플랫폼별 매핑 규칙

### 1. 강남맛집 (Gangnam Matjib)
강남맛집은 캠페인 목록 API 요청 시 `ca` 파라미터를 통해 고유 카테고리를 구분합니다.

| 원문 매개변수 (`ca`) | 내부 표준 카테고리 | 비고 |
| :--- | :--- | :--- |
| `2005` (맛집) | **음식점** | |
| `2010` (뷰티) | **뷰티**   | |
| `2015` (숙박) | **숙박**   | |
| `2020` (문화) | **문화**   | |
| `2025` (배달) | **배달**   | |
| `2030` (포장) | **포장**   | |
| `2035` (기타) | **기타**   | |

*구현 위치: `crawler/category_mapper.py` 내 `GANGNAM_CATEGORY_MAP`*

### 2. 리뷰플레이스 (Review Place)
리뷰플레이스는 '지역' 대분류 내에서 `ct2` 파라미터를 통해 세부 카테고리를 필터링합니다.

| 원문 매개변수 (`ct2`) | 내부 표준 카테고리 | 비고 |
| :--- | :--- | :--- |
| `맛집` | **음식점** | |
| `카페/베이커리` | **음식점** | '음식점'으로 통합 |
| `뷰티/건강` | **뷰티** | |
| `운동/스포츠` | **운동** | |
| `숙박` | **숙박** | |
| `문화/체험` | **문화** | |
| `생활/편의` | **생활** | |
| `시크릿쇼퍼` | **기타** | |
| `기타` | **기타** | |

*구현 위치: `crawler/category_mapper.py` 내 `REVIEWPLACE_CATEGORY_MAP`*

### 3. 리뷰노트 (Review Note)
리뷰노트는 API 응답 데이터 내 `category.title` 필드에 원본 카테고리를 명시적으로 제공합니다.

| API 응답 필드 (`category.title`) | 내부 표준 카테고리 | 비고 |
| :--- | :--- | :--- |
| `맛집` | **음식점** | |
| `뷰티` | **뷰티** | |
| `반려동물` | **반려동물** | |
| `식품` | **식품** | |
| `여행` | **여행** | |

*(위 목록에 포함되지 않는 '디지털', '기타' 등의 값은 후술할 '키워드 대체 추론' 방식을 거칩니다.)*

*구현 위치: `crawler/category_mapper.py` 내 `REVIEWNOTE_CATEGORY_MAP`*

---

### 4. 디너의여왕 (Dinner Queen)
디너의여왕은 별도의 식별 파라미터가 없으나, 접속 URL 경로에 따라 대분류(TASTE, BEAUTY 등)가 고정되어 있습니다. 이를 내부 카테고리와 매칭합니다.

| 분류 경로 이름 | 내부 표준 카테고리 | 비고 |
| :--- | :--- | :--- |
| `taste` | **음식점** | |
| `beauty` | **뷰티** | |
| `life` | **생활** | |
| `travel` | **여행** | |

*구현 위치: `crawler/category_mapper.py` 내 `DINNERQUEEN_CATEGORY_MAP`*

### 5. 서울오빠 (Seoul Oppa)
서울오빠는 방문형 캠페인(`cat=377`) 하위에 위치한 전용 카테고리 고유 식별자(`cat`) ID를 순회 방식으로 수집하여 매핑합니다. 

| 원문 매개변수 (`cat`) | 내부 표준 카테고리 | 비고 |
| :--- | :--- | :--- |
| `378` (맛집) | **음식점** | |
| `379` (여행/숙박) | **숙박** | |
| `380` (뷰티/패션) | **뷰티** | |
| `381` (문화/생활) | **문화** | |
| `382` (테이크아웃) | **포장** | |
| `510` (스포츠/레저) | **레저** | |

*구현 위치: `crawler/category_mapper.py` 내 `SEOULOPPA_CATEGORY_MAP`*

## 키워드 기반 대체 추론 (Fallback Extraction)
리뷰노트처럼 매핑 사전에 명시되지 않은 분류가 반환될 경우, 캠페인의 **제목**과 **제공 내역**의 텍스트를 분석하여 카테고리를 강제 배정합니다.

주요 키워드 목록:
- **뷰티**: 에스테틱, 미용실, 헤어, 염색, 네일, 왁싱, 피부관리, 마사지 등
- **숙박**: 호텔, 펜션, 풀빌라, 캠핑 등
- **반려동물**: 강아지, 고양이, 애견 등
- **유아동**: 키즈, 어린이, 육아 등
- **운동**: 헬스, pt, 요가, 필라테스 등
- **레저**: 스키, 보드, 골프, 방탈출 등
- **문화**: 전시, 공연, 연극, 원데이클래스, 공방 등
- **음식점**: 오마카세, 회, 파스타, 디저트, 카페 등
- **기타**: 상기 어느 곳에도 속하지 않을 시 최종 할당

*구현 위치: `crawler/category_mapper.py` 내 `extract_category_by_keyword()`*

## 운영 및 유지보수 가이드
체험단 사이트의 개편으로 인해 카테고리 구조나 파라미터가 변경된 경우, **반드시 `crawler/category_mapper.py` 모듈만 수정**하여 크롤러 로직에 14종 데이터가 안정적으로 전달되도록 관리합니다.
